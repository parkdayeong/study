createElement

지정한 tagName의 html요소를 생성합니다

let element = document.createElement('p');

appendChild

마지막 자식요소로 요소추가 삽입할 요소만 매개변수로 제공하면된다.

const = li = document.createElement('li');
li.textContent = "hello";
document.querySelector('ul').appendChild(li);

insertBefore

매개변수를 두개 받습니다. 첫번재 매개변수는 삽입할요소, 두번재 매개변수는 삽입할 위치를 정하는 요소

const firstItem = document.querySelector('.first-item');
const li = document.createElement ('li');
li.textContent ="hi";
document.querySelector('ul').insertBefore(li, firstItem)

remove

해당요소 제거

parentNode

지정된 노드의 부모를 반환

removeBtn.addEventListener('click', function(event){ console.log('remove click'), event.target event.target.parentNode.remove(); })
