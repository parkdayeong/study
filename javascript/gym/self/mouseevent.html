마우스 클릭 이벤트:

웹 페이지에 버튼이나 링크를 생성하고, 해당 요소를 클릭할 때마다 콘솔에 메시지를 출력하는 코드를 작성하세요.

마우스 오버 및 아웃 이벤트:

웹 페이지에 이미지를 추가하고, 이미지 위에 마우스를 가져가면 이미지가 커지고, 마우스를 이미지 바깥으로 이동하면 이미지가 작아지도록 코드를 작성하세요.

드래그 앤 드롭:

웹 페이지에 여러 개의 이미지를 추가하고, 이미지를 드래그하여 원하는 위치로 이동할 수 있는 인터랙티브한 환경을 만들어 보세요.

마우스 휠 스크롤 이벤트:

특정 요소에 마우스 휠을 사용하여 스크롤 이벤트를 처리하고, 스크롤 방향에 따라 콘텐츠가 이동하도록 코드를 작성하세요.

오른쪽 버튼 클릭 방지:

웹 페이지의 모든 오른쪽 마우스 클릭을 차단하여 컨텍스트 메뉴가 나타나지 않도록 코드를 작성하세요.

이미지 확대/축소:

웹 페이지에 이미지를 추가하고 마우스 휠을 사용하여 이미지를 확대 또는 축소할 수 있는 코드를 작성하세요.

클릭한 마우스 좌표 표시:

웹 페이지를 클릭할 때마다 마우스 좌표 (X, Y)를 화면에 표시하는 코드를 작성하세요.

마우스 오버 효과:

웹 페이지 요소에 마우스를 가져가면 배경색이 바뀌거나 특정 애니메이션 효과가 나타나도록 코드를 작성하세요.
