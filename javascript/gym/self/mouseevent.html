mouseevent

  • click: 요소를 마우스로 클릭했을 때 발생합니다. 더블 클릭은 두 번 클릭하는 것을 의미하며, 각각의 클릭은 별도의 click 이벤트로 처리됩니다.
  • mousedown: 마우스 버튼을 누를 때 발생합니다.
  • mouseup: 마우스 버튼을 눌렀다 놓을 때 발생합니다.
  • mousemove: 마우스 커서가 요소 내에서 움직일 때 발생합니다.
  • mouseover: 마우스 커서가 요소 위로 이동할 때 발생합니다.
  • mouseout: 마우스 커서가 요소를 벗어날 때 발생합니다.
  • mouseenter: 마우스 커서가 요소 내부로 들어올 때 발생하며, 버블링이 일어나지 않습니다.
  • mouseleave: 마우스 커서가 요소 내부를 떠날 때 발생하며, 버블링이 일어나지 않습니다.
  • contextmenu: 마우스 오른쪽 버튼을 클릭하거나 컨텍스트 메뉴를 표시하려고 할 때 발생합니다.
  • dblclick: 요소를 더블 클릭할 때 발생합니다.
  • wheel: 마우스 휠을 사용하여 스크롤할 때 발생합니다.
  • dragstart: 드래그 작업이 시작될 때 발생합니다.
  • dragend: 드래그 작업이 종료될 때 발생합니다.
  • dragover: 요소 위로 드래그 중인 아이템을 가져올 때 발생합니다.
  • dragenter: 요소에 드래그 중인 아이템이 들어갈 때 발생합니다.
  • dragleave: 요소에서 드래그 중인 아이템이 나갈 때 발생합니다.
  • drop: 드래그된 아이템을 요소에 드롭할 때 발생합니다.
  • mousedown, mousemove, mouseup의 조합으로 드래그 및 드롭을 구현할 수 있습니다.

유용한 이벤트 속성&메서드

  • event.clientX, event.clientY
  • event.pageX, event.pageY
  • event.target.getBoundingClientRect()
